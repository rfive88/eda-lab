# data/nangate45/

The Nangate45 open cell library LEF set, copied from
`third_party/openroad/test/Nangate45/` at the pinned SHA. The test suites
use `Nangate45_tech.lef` (tech) and `Nangate45_stdcell.lef` (cells); the
remaining LEF variants (lvt, fakeram macros, pads, ...) came along with
the copy and are currently unused. See `data/README.md`.
